## 🛠 Technology Blueprint

Hutch Turbo is built on a modern hybrid architecture, combining native performance with flexible web-based UI technologies.

- **Frontend**: High-performance WebView utilizing HTML5, CSS3, and JavaScript with a custom-built HUD interface.
- **Backend Core**: Kotlin 1.9+ leveraging **Coroutines** for non-blocking network operations.
- **Service Layer**: Foreground Services with specialized notification management for Android 13+ compatibility.
- **Bridge Architecture**: A secure, custom-defined `TurboBridge` for zero-latency communication between JS and Native layers.

## 📦 Quick Start

### Prerequisites
- Android Studio Hedgehog or newer
- JDK 17
- Android Device/Emulator (API 24+)

### Installation Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/NiwanthaSandaruwan-Developer/Hutch-Network-Booster-Android-Application.git
   ```

2. **Open in Android Studio**
   - Point to the `Hutch_Turbo` directory.
   - Sync Project with Gradle Files.

3. **Build & Run**
   - Select your device and hit `Shift + F10`.
